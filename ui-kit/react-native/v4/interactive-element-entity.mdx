Following are the different elements possible

1. [Label Element](/ui-kit/react-native/v4/interactive-label-element) - shows fixed text in an Interactive message
2. [TextInput Element](/ui-kit/react-native/v4/interactive-textinput-element) - shows a component with the ability to take text input
3. [DropDown Element](/ui-kit/react-native/v4/interactive-dropdown-element) - shows a list of options to choose one from
4. [Checkbox Element](/ui-kit/react-native/v4/interactive-checkbox-element) - shows a list of checkboxes, and can select multiple
5. [Single Select Element](/ui-kit/react-native/v4/interactive-single-select-element) - show a list of options to choose one from, a different way to show radio buttons
6. [Radio Button Element](/ui-kit/react-native/v4/interactive-radio-button-element) - shows a list of radio buttons, to choose one answer from
7. [Button Element](/ui-kit/react-native/v4/interactive-button-element) - shows an interactive button component, that can perform different types of actions defined inside class ActionEntity



ElementType specifies the type of interactive element.

| Name         | Description                       |
| :------------ | :--------------------------------- |
| label        | An unmodifiable text description. |
| text         | A text input field.               |
| dropdown     | A dropdown menu.                  |
| checkbox     | A checkbox input.                 |
| radio        | A radio button input.             |
| button       | An interactive button.            |
| singleSelect | A single-selection menu.          |
