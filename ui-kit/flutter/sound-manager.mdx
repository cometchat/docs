---
title: "Sound Manager"
---

<Info>
  The **SoundManager** is a helper class responsible for managing and playing
  various types of audio in the CometChat UI Kit. This includes sound events for
  incoming and outgoing messages and calls.
</Info>

---

## Methods

<CardGroup cols={1}>
  <Card title="Play Sound" icon="volume-high">
    This method plays pre-defined or custom sounds based on user interactions. If no custom sound file is provided, the default sound is played.

    <Tabs>
      <Tab title="Dart">
        ```dart
        // Play sound with default sound:
        SoundManager().play(sound: Sound.incomingMessage); //To play default incoming message sound
        SoundManager().play(sound: Sound.outgoingMessage);//To play default outgoing message sound
        ```
      </Tab>
    </Tabs>
  </Card>
  <Card title="Stop Sound" icon="volume-xmark">
    This method stops any sound currently being played for incoming and outgoing calls and messages.

    <Tabs>
      <Tab title="Dart">
        ```dart
        SoundManager().stop();
        ```
      </Tab>
    </Tabs>
  </Card>
</CardGroup>

<Warning>
  CometChat sound will behave differently with different types of OS in case of
  background playing.
</Warning>

---

## Usage
Here is how to play a custom sound using the `SoundManager`.
<Tip title="How to use SoundManager">
   By using the SoundManager, you can enhance the user experience in your chat application by
  integrating audible cues for chat interactions.
</Tip>
  <Tabs>
    <Tab title="Dart">
      ```dart
      //Play sound with custom sound
      SoundManager().play(sound: Sound.outgoingMessage, customSound: "assetPath"); //To play a custom message sound
      ```
    </Tab>
  </Tabs>


---

## Default Sound Assets

The table below lists the various `Sound` enum cases and the corresponding default asset paths.

| Sound                    | Asset Path                         |
| ------------------------ | ---------------------------------- |
| `incomingMessage`        | `assets/sound/incoming_message.wav`|
| `outgoingMessage`        | `assets/sound/outgoing_message.wav`|
| `incomingMessageFromOther`| `assets/sound/incoming_message.wav`|
| `outgoingCall`           | `assets/sound/outgoing_call.wav`   |
| `incomingCall`           | `assets/sound/incoming_call.wav`   |
