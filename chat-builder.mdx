	<Steps>
		<Step title="Copy or download">Download the generated project or copy the component bundle.</Step>
		<Step title="Install deps">Run <code>npm install</code> / <code>yarn</code> to install CometChat + peer dependencies.</Step>
		<Step title="Provide credentials">Insert your <code>APP_ID</code>, <code>REGION</code>, and authentication logic.</Step>
		<Step title="Test locally">Run the dev server and confirm login, messages, presence, media upload.</Step>
		<Step title="Customize incrementally">Tweak styles, swap icons, override component props as needed.</Step>
	</Steps>

	<Callout type="info" title="Safe Customization">
		You can safely refactor layout files; avoid editing generated core logic utilities unless you need deep behavior changes.
