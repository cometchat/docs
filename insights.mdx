## Why It Matters

* Gain detailed statistics about user interactions with CometChat services.
* Measure user engagement using various parameters.
* Identify churn patterns and take corrective actions.
* Define and analyze conversational funnels using AI-powered insights.
* Classify and measure user personalities based on conversation styles.
* Optimize chat experiences and enhance user engagement through data-driven decision-making.

## Users & Groups

High‑level engagement & population health.

<AccordionGroup>
	<Accordion title="User Engagement Metrics">
		<ul>
			<li>Overview and trends: Displays user activity trends over time.</li>
			<li>Top 10 users: Identifies the most active users based on chat activity.</li>
			<li>Top 10 groups: Lists the most engaged chat groups.</li>
			<li>Average group size: Useful for social and community apps to determine group engagement levels.</li>
			<li>Concurrent users heatmap: Shows online activity per hour on a given day to determine user activity trends.</li>
			<li>OS usage: Indicates number of users per platform (iOS, Android, Web).</li>
		</ul>
	</Accordion>
	<Accordion title="Growth & Engagement Metrics">
		<ul>
			<li>New users created (daily/monthly): Tracks growth trends in user registrations.</li>
			<li>Time to first message: Measures user engagement speed post-signup.</li>
			<li>Users who reacted the most: Shows most active and influenced users from a Reaction perspective.</li>
			<li>Users who received the most Reactions: Shows most influential users.</li>
			<li>Online frequency distribution: Shows how often users come online in a day, indicating their engagement level.</li>
			<li>New friends Count: Tracks relationship-building activities in social or dating apps.</li>
			<li>Active group count: Measures group engagement over time.</li>
			<li>Online users per day: Shows daily active user trends.</li>
		</ul>
	</Accordion>
	<Accordion title="Churn & Retention Analysis">
		<ul>
			<li>User retention analysis: Tracks number of users who have some activity in the last 30 days to assess churn trends.</li>
			<li>Cohort analysis of user abandonment: Identifies when users drop off and helps address engagement gaps and abandonment analysi.</li>
			<li>User abandon analysis funnel: Categorizes user churn based on activity type (e.g., never sent a message, sent messages but never made a call, etc.) for abandonment analysis.</li>
			<li>Group churn per day: Tracks the number of groups which don't have any messages in the last 30 days to assess group churn trends.</li>
		</ul>
	</Accordion>
</AccordionGroup>

<a id="messaging" />
## Messaging Insights

Conversation flow quality & content dynamics.

<AccordionGroup>
	<Accordion title="Overview & Trends">
		<ul>
			<li>Messaging activity: Tracks overall message sent, delivered and read numbers per day.</li>
			<li>Messages sent heatmap: Displays data to indicate hours when users perfer to send messages.</li>
			<li>Messages delivered heatmap: Displays data to indicate hours when users typically receive messages.</li>
			<li>Messages sent heatmap: Displays data to indicate hours when users perfer to read messages.</li>
			<li>Messages by conversation type: Breaks down messages into one-on-one and group conversations.</li>
			<li>Average messages across conversations: Measures engagement levels in terms of the number of messages sent in conversations.</li>
			<li>Message types sent : Identifies trends in text, images, video, and other media types.</li>
			<li>Thread messages: Shows the number of conversations that generated threaded discussions.</li>
			<li>Average word and character Count: Measures the verbosity of messages.</li>
		</ul>
	</Accordion>
	<Accordion title="Growth & Engagement Metrics">
		<ul>
			<li>New conversations per day: Indicates overall chat engagement growth.</li>
			<li>Active conversation distribution: Shows a distribution of the number of ongoing conversations accross users.</li>
			<li>Message engagement frequency: Tracks how often in terms of days do users send messages over time.</li>
			<li>User messaging behavior: Differentiates between active and passive users from their messaging behaviour.</li>
			<li>Number of messages before first image: Measures the transition from text to media engagement.</li>
			<li>Engagement progression by message type: Measures the transition from text to an alternative message type (excluding images).</li>
			<li>Average first response time: Measures response speed in a new conversations.</li>
			<li>Average response time per day: Measures response speed in a conversations per day.</li>
			<li>Chat to calls conversions: Measures the number of conversations which resulted in a call signifying an increase in engagement.</li>
			<li>Messages vs voice vs video: Measures the proportion of messages, voice calls, and video calls to measure the engagement patterns.</li>
			<li>Month over month active conversations: Compares the number of active conversations in a month compared to the previous one.</li>
			<li>Pre-call messaging: Measures the number of messages exchanged before a call.</li>
			<li>Number of mentions: Measures the number of mentions across conversations.</li>
			<li>Reactions count: Measures the number of reactions across conversations.</li>
		</ul>
	</Accordion>
	<Accordion title="Retention & Churn Metrics">
		<ul>
			<li>Number of conversation churns: Measures the number of conversations without any new messages the last 30 days indicating an eminent churn.</li>
			<li>Messages before conversation churn: Measures the number of messages exchanged before conversation churns.</li>
			<li>Conversation churn time distribution: Measures the number of days after which conversations churn.</li>
			<li>Declining conversation count: Measures the number of conversations with a consistently declining message volume.</li>
		</ul>
	</Accordion>
</AccordionGroup>

## Voice & Video Calling Insights

Real‑time engagement patterns.

<AccordionGroup>
	<Accordion title="Call Metrics">
		<ul>
			<li>Call duration: Measures the total call duration across all calls for a given day.</li>
			<li>Call participation: Measures the average number of users in a call across all calls on a given day.</li>
			<li>Concurrent calls heatmap: Displays data to indicate hours when users perfer to make calls.</li>
			<li>Call duration distribution: Measures the call duration pattern across calls.</li>
			<li>Calls per day: Measures the total calls per day for a given day.</li>
			<li>Calling engagement frequency: Tracks how often in terms of days do users make calls.</li>
		</ul>
	</Accordion>
</AccordionGroup>
